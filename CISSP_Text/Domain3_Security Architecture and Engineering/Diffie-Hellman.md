
Diffie-Hellman is an algorithm for securely exchanging a common key between two parties over an insecure communication channel.
In fact, the parties do not share a common key, but rather a value used to generate a common key.

ディフィー・ヘルマンとは、安全でない通信路を介して、2者間で共通鍵を安全に交換するためのアルゴリズムです。
実際には共通鍵ではなく、共通鍵を生成するための値を共有します。
