
The salt is a random value added to a password before it is hashed by the operating system. 
The salt is then stored in a password file with the hashed password. 
This increases the complexity of cryptanalytic attacks.


ソルトとは、オペレーティングシステムによってハッシュ化される前にパスワードに追加されるランダムな値です。
ソルトはハッシュ化されたパスワードと共にパスワードファイルに保存されます。
これにより、暗号解読攻撃の複雑さが増します。
