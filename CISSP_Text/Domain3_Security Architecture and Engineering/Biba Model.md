
The Biba Model is a state transition system describing a set of access control rules designed to ensure data **integrity**. 
Data and subjects are grouped into ordered levels of integrity. 
The model is designed so that subjects may not corrupt data in a level ranked higher than the subject, or be corrupted by data from a lower level than the subject.

Bibaモデルは、データの**完全性**を確保するために設計された一連のアクセス制御ルールを記述した状態遷移システムです。
データとサブジェクトは、整合性の順序付けられたレベルにグループ化されます。
このモデルは、サブジェクトが自身よりも上位レベルのデータを破損したり、サブジェクトよりも下位レベルのデータによって破損したりしないように設計されています。
