
SaaS is a system for providing software as a service.
Typical examples include Google Workspace (including Gmail and Google Drive) and Microsoft 365.

PaaS is a system for providing platform as a service.
A typical example is Google App Engine (which runs programs in Google's server environment and allows you to publish web applications).

IaaS is a system for providing infrastructure as a service.
Typical examples are AWS EC2 and Google Compute Engine.

CaaS is a system for providing advanced computing as a service.
Typical examples are Amazon Elastic Kubernetes Service (Amazon EKS), Google Kubernetes Engine (GKE), and Azure Kubernetes Service (AKS).


SaaSは「ソフトウェアをサービスとして提供する」仕組みです。
代表例はGoogle Workspace（GmailやGoogle Driveなど）やMicrosoft 365です。

PaaSは「プラットフォームをサービスとして提供する」仕組みです。
代表例はGoogle App Engine（Googleのサーバー環境でプログラムを実行し、Webアプリケーションの公開ができる）です。

IaaSは「インフラストラクチャをサービスとして提供する」仕組みです。
代表例はAWS EC2、Google Compute Engineです。

CaaSは「高度なコンピューティングをサービスとして提供する」仕組みです。
代表例はAmazon Elastic Kubernetes Service (Amazon EKS), Google Kubernetes Engine (GKE), Azure Kubernetes Service (AKS) です。
