
Data State is a classification of data based on where it is stored and the context in which it is used.
Data State can be [[Data at rest]] (on a drive or other storage medium), [[Data in use]] and thus [[Data in motion]] or a buffer and often decrypted for use, or in motion over a network.

データの状態（Data State）とは、データが保持されている位置や利用される状況に基づき、データを分類したものです。
データの状態は保存中（ドライブまたはその他のストレージメディア上）、使用中（メモリまたはバッファ内、使用時に復号化されることが多い）、またはネットワーク経由で移動中のいずれかです。
